API Basejump: URL Shortener Microservice


Usage tips


-As a parameter string of URL passed and JSON data generated with keys 'original_url' and 'short_url'.

-If short URL is passed as a parameter then it should be reirected to original URL.

-There will be error if URL string provided is not valid or short URL does not match with generated one.

Example of how URL string should be passed as a parameter:


https://urlsh.herokuapp.com/new/http://www.bbc.com

Generated JSON data as a result:


{"original_url":"http://www.bbc.com","short_url":"https://urlsh.herokuapp.com/8"}

Passing this short URL as a parameter:


https://urlsh.herokuapp.com/8

Will redirect to this original URL:


http://www.bbc.com
