API Basejump: Timestamp microservice


Usage tips


-As a parameter string is passed which contains name of the month, year ( numbers with 4 characters more than 1000), and date ( numbers between 1 and 31).

-Month, year, date are separated with any symbols other than letters, numbers and space.

-Or timestamp is passed as a parameter (numbers with 13 characters).

-In case if parameter is a string or timestamp then Unix timestamp and natural language form of the date is returned in JSON format.

-Else the value of 'unix' and 'natural' keys of JSON data will be null

Example:


https://timestamp-microser.herokuapp.com/December%1993&3

https://timestamp-microser.herokuapp.com/754858800000

Output:


{"unix":754858800000,"natural":"December 3, 1993"}
